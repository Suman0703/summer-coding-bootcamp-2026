# Day 25 - Linked List Insertion Operations (C++)

## 📌 Overview

This folder contains C++ implementations of various **Linked List insertion operations**. Each insertion technique is implemented separately for better understanding and practice.

## 📂 Files

| File Name | Description |
|-----------|-------------|
| `Linkedlist.cpp` | Complete Linked List implementation with basic operations. |
| `insertion_before.cpp` | Insert a new node before a specified node. |
| `insertion_after.cpp` | Insert a new node after a specified node. |
| `insertion_atposition.cpp` | Insert a new node at a given position in the linked list. |

---

## 📚 Concepts Covered

- Singly Linked List
- Dynamic Memory Allocation
- Node Creation
- Pointer Manipulation
- Traversing a Linked List
- Insertion Before a Node
- Insertion After a Node
- Insertion at a Specific Position

---

## 🎯 Learning Outcome

After completing these programs, you will understand:

- How linked lists store data dynamically.
- How nodes are connected using pointers.
- Different techniques for inserting nodes.
- Pointer updates required to maintain list integrity.
