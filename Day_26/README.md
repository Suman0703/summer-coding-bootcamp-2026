# Day 26 - Linked List Deletion & Reversal (C++)

## 📌 Overview

This folder contains C++ implementations of various **Linked List deletion operations** along with the **reverse operation**. Each program focuses on a specific operation to strengthen understanding of pointer manipulation in a singly linked list.

## 📂 Files

| File Name | Description |
|-----------|-------------|
| `linkedlist.cpp` | Complete Linked List implementation in C++. |
| `linkedlist.c` | Complete Linked List implementation in C. |
| `deletion_at_beg.cpp` | Delete the first node of the linked list. |
| `deletion_at_end.cpp` | Delete the last node of the linked list. |
| `deletion_before.cpp` | Delete the node before a specified node. |
| `deletion_after.cpp` | Delete the node after a specified node. |
| `deletion_at_position.cpp` | Delete a node from a specified position. |
| `reverse.cpp` | Reverse the linked list. |

---

## 📚 Concepts Covered

- Singly Linked List
- Dynamic Memory Allocation

- Node Deletion
- Traversing a Linked List
- Deletion at Beginning
- Deletion at End
- Deletion Before a Node
- Deletion After a Node
- Deletion at a Specific Position
- Reversing a Linked List

## 🎯 Learning Outcomes

After completing these programs, you will understand:

- Different linked list deletion techniques.
- Safe pointer updates during node deletion.
- Handling edge cases in linked lists.
- Reversing a linked list efficiently using pointers.
- Memory management in dynamic data structures.
