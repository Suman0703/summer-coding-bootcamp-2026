# Doubly Linked List in C++

A menu-driven **Doubly Linked List** implementation in **C++** using **Object-Oriented Programming (OOP)** concepts.

## 📌 Features

* Insert at Beginning
* Insert at End
* Insert Before a Node
* Insert After a Node
* Insert at Specific Position
* Display Doubly Linked List
* Delete from Beginning
* Delete from End
* Delete Before a Node
* Delete After a Node
* Delete at Specific Position
* Edit Node Value
* Count Total Nodes
* Sort Linked List
* Reverse Display

## 🛠️ Technologies Used

* C++
* Classes and Objects
* Constructors
* Dynamic Memory Allocation (`new` / `delete`)
* Pointers
* Menu-Driven Programming

## 📚 Concepts Covered

* Object-Oriented Programming (OOP)
* Class Implementation
* Encapsulation
* Constructors
* Dynamic Memory Management
* Doubly Linked List Operations
* Pointer Manipulation

## 🎯 Learning Outcomes

This project demonstrates how to convert a procedural C implementation of a Doubly Linked List into an object-oriented C++ implementation while preserving the same logic. It provides hands-on practice with linked list operations, classes, constructors, and dynamic memory management.
