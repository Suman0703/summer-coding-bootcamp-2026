# Day 23 - C++ OOP, Templates & STL Vector

This repository contains C++ programs demonstrating important Object-Oriented Programming (OOP) concepts, templates, and Standard Template Library (STL) containers.

## Programs Included

### 1. addition_using_class.cpp
- Demonstrates addition using a C++ class.
- Shows object creation and member functions.

### 2. friend_class.cpp
- Demonstrates the Friend Class concept.
- Allows one class to access another class's private members.

### 3. friend_function.cpp
- Demonstrates Friend Functions.
- Shows how non-member functions can access private data.

### 4. Generic_Class.cpp
- Demonstrates Class Templates.
- Creates generic classes that work with multiple data types.

### 5. Generic_mixedtype_class.cpp
- Demonstrates templates with multiple data types.
- Uses more than one template parameter.

### 6. Inline_function.cpp
- Demonstrates Inline Functions.
- Improves performance by reducing function call overhead.

### 7. local_global_sro.cpp
- Demonstrates Local and Global Variables.
- Explains variable scope and lifetime.

### 8. private_datamember.cpp
- Demonstrates Encapsulation.
- Shows how private data members protect data.

### 9. Searching_tempalte.cpp
- Implements a Generic Searching program using Function Templates.
- Works with different data types.

### 10. Searching2.cpp
- Demonstrates another searching implementation in C++.

### 11. Template.cpp
- Additional examples of C++ templates.

### 12. Vector.cpp
- Demonstrates the use of the STL Vector container.
- Covers insertion, traversal, and common vector operations.
