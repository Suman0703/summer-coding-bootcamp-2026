### `RootsofQuadraticEqn.c`

This program finds the roots of a quadratic equation using the formula:

```
ax² + bx + c = 0
```

The program handles all three cases:

- Two distinct real roots
- One equal real root
- Two complex roots

**Concepts Used**

- `if-else`
- `sqrt()` from `<math.h>`
- Floating-point variables
- Mathematical formulas
