# Day 1 - Introduction to C Programming

## 📚 Overview
On Day 1, I learned the basics of the C programming language. The session focused on understanding what C is, why it is widely used, and how to write and execute a simple C program.

## 📝 Topics Covered
- Introduction to C Programming
- Features of C
- Structure of a C Program
- `#include<stdio.h>`
- `main()` function
- `printf()` function
- `return 0`
- Comments in C
  - Single-line comments (`//`)
  - Multi-line comments (`/* */`)
- Basic Output
- `for` Loop
- Printing numbers using a loop

## 💻 Practice Program

```c
#include <stdio.h>

int main()
{
    for (int i = 0; i <= 20; i++)
    {
        printf("Hello World! ");
        printf("%d\n", i);
    }

    return 0;
}
```

## 📌 Output

```
Hello World! 0
Hello World! 1
Hello World! 2
...
Hello World! 20
```

## 🎯 Key Learnings
- Understood the basic structure of a C program.
- Learned how to display output using `printf()`.
- Learned the purpose of the `main()` function.
- Used comments to improve code readability.
- Wrote a simple `for` loop to repeat a task multiple times.
